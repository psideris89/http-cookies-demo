# http-cookies-demo 🍪

This repo demonstrates the use of Http Cookies and it contains a front-end application (React.js) and a Node.js/Express.js server.

## Installation

#### To start the front-end app.

```
$ cd cookies-app
$ npm install
$ npm start
```

#### To start the server.

```
$ cd api
$ npm install
$ npm run dev
```
